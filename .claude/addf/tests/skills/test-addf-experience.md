# テスト: addf-experience

## テスト 1: 経験参照の自己整合性チェック（全スキル）

### 入力
```
/addf-experience
```

### 期待結果
- `.claude/commands/` と `.claude/addf/optional/commands/` 配下の全スキルを走査する
- 各スキルの `.exp.md` 参照を一覧化する（参照がないスキルも「経験活用の記述なし」として含める）
- 参照ファイル名がスキル自身の名前と一致することを確認する（不一致は自己参照ミスとして報告）
- 「読み込み」指示と「書き込み」指示の両方が揃っていることを確認する（片方欠けは報告）
- `@<name>.exp.md` 形式（展開されない誤解を招く書式）があれば報告する
- 問題がなければ「問題なし」と報告する

## テスト 2: 特定スキル指定

### 入力
```
/addf-experience addf-dev
```

### 期待結果
- `addf-dev.md` のみを対象に検証する
- `addf-dev.exp.md` への自己参照・読み書き両指示が揃っていることを確認し、正常と報告する

## テスト 3: 経験活用の記述がないスキルの扱い

### 入力
```
/addf-experience addf-lint
```

### 期待結果
- `addf-lint.md` に `.exp.md` 参照が存在しないことを検出する
- エラーではなく「経験活用の記述なし」として報告する（スキップして黙るのではなく、可視化する）
