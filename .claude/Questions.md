# Questions

ループ中のエージェントが「閾値割れ（確信が持てない）」と判断した事項を記録する非同期質問箱。
エージェントは止まらず質問だけ置いて次へ進み、オーナーは気が向いたタイミングで Answer を書く。
Answer が書かれたら、エージェントは次のセッション開始時（ブートシーケンス 1.5）に Plan へ反映し、該当質問を「回答済み」へ移す。

質問の書式は `.claude/Questions.example.md` の「Q例」に従う。
このファイルはオーナーとの共有チャンネルのため、リポジトリにコミットされる。
